# Security Policy

We actively support the following versions of Persian Tools with security updates:

| Version | Supported          | Node.js Required |
| ------- | ------------------ | ---------------- |
| 4.x     | :white_check_mark: | >= 14.x          |
| 3.x     | :white_check_mark: | >= 12.x          |
| < 3.0   | :x:                | N/A              |
