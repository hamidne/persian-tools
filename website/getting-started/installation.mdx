# Installation

Persian Tools is available as an npm package and supports multiple JavaScript environments including Node.js, Bun, and modern browsers.

## Package Manager Installation

<CodeGroup>
```bash npm
npm install @persian-tools/persian-tools
```

```bash yarn
yarn add @persian-tools/persian-tools
```

```bash pnpm
pnpm add @persian-tools/persian-tools
```

```bash bun
bun add @persian-tools/persian-tools
```

</CodeGroup>

## CDN Usage

For browser environments, you can use Persian Tools directly from a CDN:

<CodeGroup>
```html jsdelivr
<script src="https://cdn.jsdelivr.net/npm/@persian-tools/persian-tools@latest/build/persian-tools.min.js"></script>
```

```html unpkg
<script src="https://unpkg.com/@persian-tools/persian-tools@latest/build/persian-tools.min.js"></script>
```

</CodeGroup>
