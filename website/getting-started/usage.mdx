# Basic Usage

This guide covers the fundamental concepts and patterns for using Persian Tools effectively.

## Import Styles

Persian Tools supports multiple import styles to fit your project's needs.

### Named Imports (Recommended)

Import only the functions you need for optimal tree-shaking:

```typescript
import { numberToWords, digitsEnToFa, verifyIranianNationalId } from '@persian-tools/persian-tools';

const result = numberToWords(1234);
console.log(result); // "یک هزار و دویست و سی و چهار"
```

### Namespace Import

Import everything under a namespace:

```typescript
import * as PersianTools from '@persian-tools/persian-tools';

const result = PersianTools.numberToWords(1234);
console.log(result); // "یک هزار و دویست و سی و چهار"
```

### CommonJS

For Node.js projects using CommonJS:

```javascript
const { numberToWords, digitsEnToFa } = require('@persian-tools/persian-tools');

const result = numberToWords(1234);
```

## TypeScript Support

Persian Tools is written in TypeScript and provides complete type definitions.

```typescript
import { numberToWords, type NumberToWordsOptions } from '@persian-tools/persian-tools';

// Full type safety and autocomplete
const options: NumberToWordsOptions = {
  ordinal: false
};

const result: string = numberToWords(1234, options);
```

## Error Handling

Most functions in Persian Tools return safe, predictable values:

```typescript
import { verifyIranianNationalId, wordsToNumber } from '@persian-tools/persian-tools';

// Validation functions return boolean
const isValid = verifyIranianNationalId("0499370899"); // true

// Conversion functions handle invalid input gracefully
const number = wordsToNumber("سلام"); // Returns 0 for invalid input
```

<Tip>
  Always check return values for validation functions. They return `false` for invalid input rather than throwing errors.
</Tip>

## Common Patterns

### Chaining Operations

Combine multiple utilities for complex transformations:

```typescript
import { digitsEnToFa, addCommas, numberToWords } from '@persian-tools/persian-tools';

// Convert English digits to Persian and add commas
const formatNumber = (num: number): string => {
  return addCommas(digitsEnToFa(num.toString()));
};

console.log(formatNumber(1234567)); // "۱,۲۳۴,۵۶۷"
```

### Form Validation

Use validation utilities in forms:

```typescript
import { verifyIranianNationalId, verifyCardNumber } from '@persian-tools/persian-tools';

const validateForm = (nationalId: string, cardNumber: string) => {
  const errors: string[] = [];

  if (!verifyIranianNationalId(nationalId)) {
    errors.push('Invalid national ID');
  }

  if (!verifyCardNumber(cardNumber)) {
    errors.push('Invalid card number');
  }

  return errors;
};
```

### Data Formatting

Format data for display:

```typescript
import { digitsEnToFa, addCommas } from '@persian-tools/persian-tools';

const formatCurrency = (amount: number): string => {
  const formatted = addCommas(amount.toString());
  return `${digitsEnToFa(formatted)} ریال`;
};

console.log(formatCurrency(1500000)); // "۱,۵۰۰,۰۰۰ ریال"
```

## Performance Tips

<AccordionGroup>
  <Accordion title="Import Only What You Need" icon="leaf">
    Use named imports to ensure your bundler can tree-shake unused code:
    ```typescript
    // ✅ Good - only imports what you need
    import { numberToWords } from '@persian-tools/persian-tools';

    // ❌ Avoid - imports everything
    import * as PersianTools from '@persian-tools/persian-tools';
    ```
  </Accordion>

  <Accordion title="Reuse Compiled Patterns" icon="recycle">
    Some utilities create patterns internally. For repeated use, consider memoization:
    ```typescript
    import { isPersian } from '@persian-tools/persian-tools';

    // For validating many strings, call once per string
    const texts = ['سلام', 'hello', 'مرحبا'];
    const results = texts.map(text => isPersian(text));
    ```
  </Accordion>

  <Accordion title="Batch Operations" icon="layer-group">
    When processing multiple items, batch them to reduce overhead:
    ```typescript
    import { digitsEnToFa } from '@persian-tools/persian-tools';

    const numbers = ['123', '456', '789'];
    const converted = numbers.map(num => digitsEnToFa(num));
    ```
  </Accordion>
</AccordionGroup>
