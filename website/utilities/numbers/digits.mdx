# Digit Conversion

Convert digits between Persian (۰-۹), Arabic (٠-٩), and English (0-9) numeral systems.

## Available Functions

- `digitsEnToFa` - English to Persian digits
- `digitsFaToEn` - Persian to English digits
- `digitsArToFa` - Arabic to Persian digits
- `digitsArToEn` - Arabic to English digits
- `digitsFaToAr` - Persian to Arabic digits (rare)

## Basic Usage

```typescript
import {
  digitsEnToFa,
  digitsFaToEn,
  digitsArToFa
} from '@persian-tools/persian-tools';

// English to Persian
console.log(digitsEnToFa('123'));
// Output: "۱۲۳"

// Persian to English
console.log(digitsFaToEn('۱۲۳'));
// Output: "123"

// Arabic to Persian
console.log(digitsArToFa('٠١٢٣'));
// Output: "۰۱۲۳"
```

## API Reference

### digitsEnToFa

Convert English digits to Persian.

<ParamField path="input" type="string" required>
  String containing English digits (0-9)
</ParamField>

<ResponseField name="result" type="string">
  String with Persian digits (۰-۹)
</ResponseField>

```typescript
digitsEnToFa('Price: 1234 Toman');
// "Price: ۱۲۳۴ Toman"
```

### digitsFaToEn

Convert Persian digits to English.

<ParamField path="input" type="string" required>
  String containing Persian digits (۰-۹)
</ParamField>

<ResponseField name="result" type="string">
  String with English digits (0-9)
</ResponseField>

```typescript
digitsFaToEn('قیمت: ۱۲۳۴ تومان');
// "قیمت: 1234 تومان"
```

### digitsArToFa

Convert Arabic digits to Persian.

<ParamField path="input" type="string" required>
  String containing Arabic digits (٠-٩)
</ParamField>

<ResponseField name="result" type="string">
  String with Persian digits (۰-۹)
</ResponseField>

```typescript
digitsArToFa('٠١٢٣٤٥٦٧٨٩');
// "۰۱۲۳۴۵۶۷۸۹"
```

### digitsArToEn

Convert Arabic digits to English.

<ParamField path="input" type="string" required>
  String containing Arabic digits (٠-٩)
</ParamField>

<ResponseField name="result" type="string">
  String with English digits (0-9)
</ResponseField>

```typescript
digitsArToEn('٠١٢٣');
// "0123"
```

## Examples

### Mixed Content

Converts only the digits, preserving other characters:

```typescript
import { digitsEnToFa } from '@persian-tools/persian-tools';

console.log(digitsEnToFa('تاریخ: 2024/10/14'));
// "تاریخ: ۲۰۲۴/۱۰/۱۴"

console.log(digitsEnToFa('کد پستی: 1234567890'));
// "کد پستی: ۱۲۳۴۵۶۷۸۹۰"
```

### Display Formatting

```typescript
import { digitsEnToFa, addCommas } from '@persian-tools/persian-tools';

const formatPrice = (price: number): string => {
  const withCommas = addCommas(price.toString());
  return digitsEnToFa(withCommas);
};

console.log(formatPrice(1234567));
// "۱,۲۳۴,۵۶۷"
```

### Form Input Processing

```typescript
import { digitsFaToEn } from '@persian-tools/persian-tools';

const handlePhoneInput = (input: string): string => {
  // Convert Persian digits to English for processing
  return digitsFaToEn(input);
};

console.log(handlePhoneInput('۰۹۱۲۳۴۵۶۷۸۹'));
// "09123456789"
```

### Cleaning Mixed Arabic/Persian Text

```typescript
import { digitsArToFa, toPersianChars } from '@persian-tools/persian-tools';

const normalizePersianText = (text: string): string => {
  // First convert Arabic chars to Persian
  const persianChars = toPersianChars(text);
  // Then convert Arabic digits to Persian
  return digitsArToFa(persianChars);
};

console.log(normalizePersianText('سلام ٠١٢٣'));
// "سلام ۰۱۲۳"
```
